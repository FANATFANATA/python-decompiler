# Python-Decompiler

## EN

A simple tool for decompiling Lua scripts (.luac) using unluac or luadec.

## Supported Languages
- English
- Russian

## How to Use
- git clone https://github.com/FANATFANATA/python-decompiler
- cd python-decompiler
- Python Linux: python3 main.py
- Python Windows: py main.py
- Bash: bash: main.sh
- PowerShell: main.ps1
